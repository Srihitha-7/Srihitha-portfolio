G.Srihitha · Portfolio Website

This repository contains the source code for my personal portfolio website, showcasing my journey as a Computer Science Engineer and aspiring Machine Learning Engineer.

Features

Responsive Design:Optimized for various screen sizes including mobile and desktop.
Glassmorphism UI:Modern aesthetic using background blurs and backdrop filters.
Smooth Animations:Integrated skill bar animations for a dynamic user experience.
Pink-Purple Theme:Clean, minimal design with a light pink-purple color palette.
Multi-Page Structure:Separate pages for Home, About, Projects, and Contact.

Built With

HTML5:Semantic structure for all web pages.
CSS3:Custom styling with Flexbox, Grid, Glassmorphism, and Keyframe animations.

Project Structure

portfolio-website
├── index.html       → Home page with hero section and profile card
├── about.html       → About Me section with Skills breakdown
├── projects.html    → Featured Projects showcase
├── contact.html     → Contact form and social links
└── style.css        → Shared styles across all pages

Pages

index.html— Landing page featuring a hero section and profile card with photo.
about.html— Detailed background on education, machine learning interests, and achievements. Also includes the Skills section with progress bars.
projects.html— Showcase of featured projects including Sentiment Analysis API, Cloud ML Pipeline, and Image Classifier App.
contact.html— Communication hub with a contact form and social links.
